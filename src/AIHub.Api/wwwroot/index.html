Kịch bản kỳ vọng

  1. Mặc định vào tab Chat để trao đổi với AI.
  2. Chuyển tab Quản lý, thêm khái niệm abc.
  3. Qua tab Sản phẩm, thử lọc theo name / category để xem API đổi tham số.
  4. Ở tab Chat, hỏi: "liệt kê sản phẩm category accessories" hoặc "tạo order productId ... quantity 1" để AI gọi API.
  5. Chuyển tab Đơn hàng để theo dõi trạng thái tạo đơn.
